## Run Locally

Clone the project

```bash
  git clone https://github.com/jmutton2/3d_portfolio
```

Go to the project directory

```bash
  cd 3d_portfolio
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm run start
```
